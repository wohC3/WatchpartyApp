# WatchpartyApp
Simpler app for real time video playback sync for friends built for educational purposes. (Inspired by Watch2Gether) 


Tech stack:
  HTML, CSS, JS
  ASP.NET Core 
  SignalR 
  Sqlite
 

Features:
-Real time messaging between users.
-System notificatinos when user leaves/joins.
-Multi room support, users can use different rooms.

How to run:
-in bash: 
 	-dotnet restore
	-dotnet watch run
