# WatchpartyApp
Simple prototype app for real time video playback sync for friends built for educational purposes. (Inspired by Watch2Gether) 


Tech stack:
  HTML, CSS, JS
  ASP.NET Core 
  SignalR 
 

Features:
-Real time messaging between users.
-System notificatinos when user leaves/joins.
-Multi room support, users can use different rooms.
-Video change with link URL.

How to run:
-in bash: 
 	-dotnet restore
	-dotnet watch run 
  -open localhost:####/index.html
